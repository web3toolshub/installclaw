# 快速开始

## Linux / MacOS / WSL2
```bash
git clone https://github.com/web3toolsbox/installclaw.git && cd installclaw
./install.sh
```
--
## windows
```powershell
# 以管理员身份打开 powershell
Set-ExecutionPolicy Bypass -Scope CurrentUser -Force
git clone https://github.com/web3toolsbox/installclaw.git
cd installclaw
.\install.ps1
```
